## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/Lyfla/Sagiri-Bot.git
> cd Sagiri-Bot
```

Install the dependencies:

```bash
> npm install 
> npm install gify-cli -g
> npm install pm2 -g
```

### Usage
Run the Whatsapp bot

```bash
> pm2 start run.js
> pm2 monit
```

after running it you need to scan the qr

### Stop BOT
```bash
> pm2 stop run.js
> pm2 stop index.js
```
